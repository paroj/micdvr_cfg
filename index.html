808 #16 Keycam (Micro DVR) configurator

Load Config

  1. Insert a microSD card into the camera
  2. Press and hold the shutter button, don't release it
  3. Press the power button for about 1 second and release it
  4. Release the shutter button when the light turns off
  5. file that has been written in the root directory of the camera

Configure

Save Config

  1. Save the file to the root directory of the camera
  2. Disconnect the camera from the computer
  3. Press and hold the shutter button, don't release it
  4. Press the power button for about 1 second and release it
  5. Release the shutter button when the light turns off
  6. The camera will accept the changes and automatically delete the file
